## 📜 How to Use
1. Clone this repository:
   ```sh
   git clone https://github.com/your-username/your-repo.git
   ```
2. Navigate into the project folder:
   ```sh
   cd project-name
   ```
3. Open `index.html` in a browser or set up a local server for a better experience.
